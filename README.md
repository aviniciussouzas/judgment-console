# Judgment Console

A small interface for evaluating ideas in the age of infinite generation.

Instead of asking what AI can generate, this console asks a different question:

**what still depends on human judgment?**

---

## What it is

A conceptual interface that turns creative evaluation into a visible, structured process.

---

## Why this exists

We are no longer limited by idea generation.

In a landscape flooded with fast outputs, the real challenge shifts:

**not generating more ideas, but recognizing which ones are actually worth developing.**

Judgment Console explores this shift by focusing on evaluation instead of production.

---

## What it evaluates

- Clarity  
- Originality  
- Symbolic Density  
- Generic AI Risk  
- Cultural Specificity  
- System Potential  
- Human Weirdness  
- Memorability  

These dimensions are not meant to “measure creativity” precisely.

They are meant to make **judgment more legible**.

---

## How to use

- Evaluate a single idea or concept  
- Compare different versions of the same direction  
- Identify generic patterns in AI-generated outputs  
- Support early-stage creative decisions  

---

## Current state

**v0.1 — experimental prototype**

- criteria still evolving  
- scoring logic is illustrative, not definitive  
- current focus: refining evaluation dimensions
